Protect the images of any post or page content.
== Description ==

The Pluign adds a transparent layer over (desired) images of any post or page. So that visitors can't save that image to their local hard-drive.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.


The plugin has no settings. Read the plugin description to understand how to use it and check the plugin's index.php file, a couple of examples are added there for reference.
